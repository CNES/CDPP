Parameter #3
Name
THEMIS-B Maneuver Flag
Description
THEMIS-B maneuver status code (XYYZZZ). Values indicate if/what maneuver is being performed.  Spacecraft data will be unreliable during maneuvers.
      * Unpowered flight (Coast):
          - X=0, Y=00, Z=000            
      * Sequence Code X:
          - 0= No Segmentation
          - 1= Segmented Maneuver
      * Type Code YY:
          - 0= No Maneuver
          - 1= Axial-thrust Maneuver
          - 2= Side-thrust Maneuver
          - 3= Beta-thrust Maneuver
          - 4= Attitude Precession
          - 5= Spin-Rate Change
      * Sequence Number ZZZ:
          - 001-999= Maneuver Sequence Number
Support
SupportQuantity
Other
Parameter #4
Name
THEMIS-B RoI
Description
Region of interest. Different bits of this parameter give a general indication of the region that the spacecraft is in.  The first 18 bits are significant.  An active bit indicates the spacecraft is in a given region.  Multiple bits may be set simultaneously. The value of the bit is in parentheses in the following table.

          +---------------------------------------------------+
          | Bit 1(1)      | Earth Shadow                      |
          | Bit 2(2)      | Lunar Shadow                      |
          | Bit 3(4)      | Atmospheric absorbtion zone       |
          | Bit 4(8)      | South Atlantic Anomaly            |
          | Bit 5(16)     | Northern Auroral Zone             |
          | Bit 6(32)     | Southern Auroral Zone             |
          | Bit 7(64)     | Perigee Passage                   |
          | Bit 8(128)    | Inner & Outer Radiation Belts |
          | Bit 9(256)    | Deep Plasma Sphere                |
          | Bit 10(512)   | Foreshock Solar Wind              |
          | Bit 11(1024)  | Solar Wind Beam                   |
          | Bit 12(2048)  | High Magnetic Field               |
          | Bit 13(4096)  | Average Plasma Sheet              |
          | Bit 14(8192)  | Bowshock Crossing                 |
          | Bit 15(16384) | Magnetopause Crossing             |
          | Bit 16(32768) | Ground Based Observatories        |
          | Bit 17(65536) | 2-Day Conjunctions                |
          | Bit 18(131072)| 4-Day Conjunctions                |
          +---------------------------------------------------+
