Description
This data set contains definitive goniopolarimetry parameters (level 3A) derived from STEREO-B WAVES LFR (Low Frequency Receiver, 2.5 - 160 kHz). Goniopolarimetry data are computed asssuming: (i) incoming wave properties do not change over three successive antennas-pair measurements, (ii) signals measured by an antenna are the same on the two receiver channels, (iii) radio sources have an uniform brightness distribution.
